[[냥이의 하루, 안냥 - 사용자 메뉴얼]]

냥이의 하루,안냥은 고양이의 안구질환을 진단하고, 일지 작성을 통해 건강체크를하며 고양이에 대해서 AI와 채팅할 수 있는 웹입니다.

** 메뉴얼
아래의 순서대로 웹을 이용해보세요.
1. 회원가입
2. 로그인
3. 홈
4. 반려묘 등록
5. 반려묘 수정
6. 안구질환 진단
7. AI챗봇
8. 일지 쓰기
9. 동물병원 위치



** 상세 설명

1. 회원가입 
모든 기능은 로그인 후 이용가능합니다.
로그인은 회원가입 후 이용가능합니다.

- 사이드바에 회원가입 버튼 클릭하여 회원가입 페이지로 이동합니다.
- 아이디, 비밀번호, 비밀번호 확인, 닉네임, 이메일, 전화번호를 작성합니다.
- 비밀번호와 비밀번호 확인이 같은지 확인하는 확인버튼을 클릭해야합니다. 
- 회원가입하기 버튼을 클릭합니다. 


2. 로그인

- 사이드바에 로그인 버튼 클릭하여 로그인 페이지로 이동합니다.
- 아이디와 패스워드를 입력합니다.
- 로그인하기버튼을 클릭합니다. 


3. 홈
회원가입, 로그인, 마이페이지를 제외한 모든 기능은 사이드바에 홈버튼에서 진행됩니다.
홈 페이지에서 About창을 통해 간략한 기능과 고양이 질병 통계를 확인할 수 있습니다.

- 사이드바에 있는 홈버튼을 클릭하여 홈 페이지로 이동합니다.


4. 반려묘 등록
이 웹의 기능을 이용하려면 반려묘를 등록해야합니다. 
마이페이지에는 회원정보확인/수정, 회원탈퇴, 반려묘등록, 반려묘정보확인/수정/삭제이 가능합니다. 
여러 마리의 반려묘 등록이 가능합니다.

- 사이드바에 있는 마이페이지 버튼 클릭하여 마이페이지로 이동합니다.
- 반려묘등록 탭을 클릭합니다.
- 반려묘이름, 반려묘나이, 반려묘등록번호, 반려묘종,반려묘성별, 반려묘상태, 반려묘에게 먹이는 사료칼로리를 작성합니다.
- 작성 후 반려묘 등록하기버튼을 클릭합니다.


5. 반려묘 수정

- 반려묘정보 탭을 클릭합니다
- 수정하고싶은 반려묘를 선택합니다.
- 반려묘이름, 나이, 반려묘종, 반려묘성별, 반려묘상태, 반려묘에게 먹이는 사료칼로리를 수정할 수 있습니다.
- 수정하기 버튼을 클릭합니다.


6. 안구질환 진단

- 사이드바에 홈버튼 클릭하여 홈 페이지로 이동합니다.
- 상단메뉴 중 안구질환 진단 버튼을 클릭합니다.
- 안구질환 진단 탭에서 서비스 이용방법을 읽어보고, 예시사진 처럼 고양이의 안구사진을 찍습니다.
- 찍은 사진을 업로드하면 바로 진단합니다. (20초 이내)
- 20초 정도가 지나면 결과가 나옵니다. 업로드한 사진과 결과는 반려묘의 진단 기록 탭에서 확인 가능합니다.


7. AI챗봇

- 상단메뉴 중 AI챗봇 버튼을 클릭합니다.
- 냥박사 질문 가이드를 읽어보고, 알맞은 질문을 작성합니다.
- 전송 버튼을 클릭하면 AI에게 전달되어 대화가 가능합니다.


8. 일지 쓰기
오늘의 일지 작성 탭에서 입력 및 수정이 가능합니다. 

- 상단메뉴 중 일지 쓰기 버튼을 클릭합니다.
- 오늘의 일지 작성 탭에서 일지 작성이 가능합니다. 파트에 맞게 작성합니다.
- 작성 후 저장하기 버튼을 클릭합니다. 작성한 일지는 일지보기 탭에 저장됩니다. 
- 일지보기 탭에서 날짜를 선택한 후 해당날짜의 기록을 확인합니다.
- 통계보기 탭에서 기간별 몸무게 변화, 하루 섭취량 및 권장 사료량, 하루 음수량 및 권장 음수량을 확인합니다.
- 선택한기간내에 권장량 보다 좋지 않은 날 수를 보여주고, 만약 권장량보다 좋지 않을 경우 경고글을 확인할 수 있습니다.


9. 동물병원

- 상단메뉴 중 동물병원 위치 버튼을 클릭합니다.
- 검색하고 싶은 지역과 세부지역을 선택합니다.
- 선택한 지역에서 지도와 표를 통해 위치, 주소 ,운영시간, 전화번호를 확인합니다.
